# NAVI Herramientas

Sistema de Gestión de Herramientas, Activos Operativos, Hojas de Vida, Inventario y Mantenimiento.

## Proyecto

Nombre funcional:

NAVI Herramientas

Nombre técnico:

Navitrans.ToolsAssets.Management

## Stack tecnológico

- .NET 8 Web API
- Blazor Server
- PWA
- SQL Server
- Docker
- MinIO
- Hangfire
- JWT / Microsoft Entra ID

## Estructura

- src: proyectos principales
- tests: pruebas unitarias e integración
- database: scripts y migraciones
- docs: documentación funcional y técnica
- docker: configuración de contenedores
